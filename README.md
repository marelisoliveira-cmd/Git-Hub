# Git-Hub
Aprendendo a mexer no Git Hub 
Durante  a aula aprendemos a criar um repósitório, adicionar um REDME (cria e nunca edita dependencias), escolher visibilidade, adicionar licença (podendo ser exibido, mas não pode ser editado; você escolhe se tem que ser pago para ver; se pode visualizar, mas não editar. Brange, sendo uma subpasta. Commits históricos das atualizações. Codesspaces (espaço de trabalho no vs code temporário). No commit coloca a descrição do que foi feito na edição. Aprender a colocar a escrita Git clone e o link no visual code ( Primeiro vai no code e copia o link ). Extensões markeplace ficoi meio vago o aprendizado, livesshare pratica que é uma ferramenta de colaoração de código também e código em conjunto também ficou vago. Aprendemos tamém que da para dividir a tela no visual code mais seria interessante dar mais uma revisada
